# CH2CalculatorProject
Spring 본캠프 계산기 과제

APP: main 메서드 위치
Calculator: 계산기 객체
OperatorType: enum 

## 프로그램의 큰 흐름
첫번째 숫자 입력 -> 두번째 숫자 입력 -> 연산자 입력 -> 가장 먼저 저장된 값 삭제 여부 -> 입력받은 값보다 큰 결과 출력 ->
연산자별 계산 이력 출력 -> 종료

## 예외처리
연산할 때 숫자가 아닌 걸 입력한 경우, 정해진 연산자가 아닌 걸 입력한 경우, 결과 출력할 때 입력받은 값이 숫자가 아닌 경우
